The **sensors(8)** is script that displays temperatures and other power related information on FreeBSD.

### Instalation

```
mkdir -p ~/bin
fetch -o ~/bin/sensors https://raw.githubusercontent.com/vermaden/sensors/master/sensors
chmod +x ~/bin/sensors
```
